在Ubuntu上编译安装Python3最新版

· 2020-09-18 ·

Python3是一个非常好的跨平台语言，众所周知，py3无法通过包管理器(apt、yum)等安装最新版，本文介绍编译安装的方式。

准备环境

为保障正常执行，请使用root用户执行命令
安装要使用的软件包

sudo apt-get install build-essential checkinstall
sudo apt-get install libreadline-gplv2-dev libncursesw5-dev libssl-dev \
    libsqlite3-dev tk-dev libgdbm-dev libc6-dev libbz2-dev
sudo apt-get install wget

下载&解压&编译

  1. 前往Python官网获取新版本
  2. 下载安装包
wget https://www.python.org/downloads/release/python-385/
  1. 解压安装包
tar -zxvf Python-3.8-5.tgz
  1. 授予运行权限
chmod +x *
  1. 配制&编译
./configure  --prefix=/usr/local
make&&sudo make install

默认情况下，使用python3命令即可调用python3，pip3同理

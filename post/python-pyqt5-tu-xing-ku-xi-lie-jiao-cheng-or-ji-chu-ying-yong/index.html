Python-PyQt5 图形库系列教程 | 消息提示框！

· 2020-03-16 · # Python # PyQt5

前言

在上一章你已经学会了修改窗体的标题，图标，大小以及位置，接下来我们来学习一些基础的方法吧！

提示框

简介

有时候我们需要给用户提供必要的提示，我们就需要添加提示框

需要的方法/函数

  • QToolTip.setFont()方法，设置字体
  • QFont()方法，用于表示字体(str,num)->字体名，字体大小
  • self.setToolTip()方法设置背景提示，参数可为富文本。
  • btn = QPushButton('Button', self)实例化一个按钮
  • btn.setToolTip(str)方法，为一个按钮添加提示框，参数可为富文本。
  • btn.resize(btn.sizeHint())调整按钮大小，sizeHint()方法为默认大小
  • btn.move()方法，调整按钮的位置，以窗口的左上角为原点

源码

#!/usr/bin/python3
# -*- coding: utf-8 -*-

import sys
from PyQt5.QtWidgets import QWidget,QToolTip,QPushButton,QApplication
from PyQt5.QtGui import QFont

class My_tooltip(QWidget):
    def __init__(self):
        super(My_tooltip, self).__init__()

        self.initUI()

    def initUI(self):
        QToolTip.setFont(QFont('SansSerif',10))
        #设置字体为10px的SansSerif字体

        self.setToolTip('This is a <b>ToolTip</b>')
        #创建背景提示框，区别请见图片

        btn = QPushButton('Button',self)
        #创建一个按钮btn
        btn.setToolTip('This is a <b>ToolTip2</b>')
        #为‘btn’按钮添加提示框
        btn.resize(btn.sizeHint())
        #设置按钮大小，这里使用默认大小
        btn.move(50,50)
        #移动到坐标系(控件)的(50,50)位置

        self.setGeometry(300,300,300,200)
        #位置及大小设置
        self.setWindowTitle('ToolTips')
        #设置标题
        self.show()
        #初始化

if __name__ == '__main__':
    app = QApplication(sys.argv)
    ex = My_tooltip()
    sys.exit(app.exec_())

CQHTTP

下载

去Github下载最新版的插件https://github.com/richardchien/coolq-http-api/releases

“安装”

将你得到的io.github.richardchien.coolqhttpapi.cpk文件放入酷Q的app文件夹内.

配置

首先，打开一次酷Q



全部选“是”即可
打开data\app\io.github.richardchien.coolqhttpapi\config\{登入QQ}.json
修改以下两行

    "ws_reverse_url": "ws://127.0.0.1:8888/ws/",
    "use_ws_reverse": true,

如果您有疑惑可以参考Nonebot官方
如果您使用Linux，您可以不进行Nonebot的安装部分

Nonebot

下载

去Github下载最新版或者发布版https://github.com/Lparksi/bot

解压使用

下载Python

https://www.python.org/

注意勾选 ADD TO PATH

安装

在控制台(cmd,PS,ssh)依次输入

python -m pip install --upgrade pip
pip install nonebot
pip install requests
pip install aiohttp
pip install pymysql


配置

修改config.py文件

from nonebot.default_config import *
from datetime import timedelta
import re

HOST = '0.0.0.0'
PORT = 8888

SUPERUSERS = {2726043636}
COMMAND_START = {'', '/', '!', '／', '！','.'}
SESSION_EXPIRE_TIMEOUT = timedelta(minutes=2)
SESSION_RUN_TIMEOUT = timedelta(seconds=10)

SUPERUSERS 中的数字改为您机器人的主人的qq即可
在终端输入python bot.py即可运行

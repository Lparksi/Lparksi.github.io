Flask | 起步

· 2020-04-28 · # Python # Flask

Flask，一个python web框架，精巧，但强大。

前言

作为api的后端，python无疑是极好的语言，我们熟知的网站(YouTube，知乎等)都是以python为后端的，使用python等高级语言可以快速完成目标，作为初学者，同步的Falsk框架无疑是比较亲民的、

安装

pip3 install flask

起步

导入包

起步篇只使用基本的Flask模块

from flask import Flask

指定名称

将app作为Flask类的名字

app = Flask(__name__)

绑定目录

将**"/"根目录**绑定

@app.route('/')
def home():
    return "hell oword!", 200
  1. 使用app.rote 装饰器绑定函数，第一个参数为绑定的目录
  2. return 第一个值是内容(html,json...),第二个是状态码

运行

if __name__ == '__main__':
    app.run()
  1. 这是面向对象编程的函数入口
  2. 当py文件被直接运行而不是当作包导入是，执行函数
  3. app.run(host=127.0.0.1, port=5000)

测试


注意，用浏览器访问，会自动获取“/favicon.ico”站点图标

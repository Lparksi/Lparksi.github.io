网站优化

· 2020-04-09 ·

简介

我和你们一样，是一个新手站长，为了让我们的网站快速收录，增加索引，从而增加访客甚至创收。

百度收录的问题

新站

如果你是刚刚创立的船新站点，此时提交百度站长，百度会在大约几天后收录你的主页，进入考察期
如果你顺利通过了审核期，百度会将之前以及以后爬取到的链接编入索引。

动态链接

百度对链接中的？ & 等动态分隔符非常敏感，如果你的文章链接中含有此链接，请考虑 伪静态
注意：采用Hexo等生成的都是静态网站，可以直接忽略这一条

内容重复

如果相似/相同的内容在你的网站中反复出现，百度会停止收录重复的内容
如果网络上有雷同内容，百度也有可能不收录您的文章

解决方案

降低相似度，适当借鉴，要有自己的创意

收录方式

手动提交

收录速度快，效率低

自动推送

效率较高，有可能被用户浏览器拦截

Sitemap

由自动爬虫进行，您无需操心，您只需要提供合法的Sitemap文件链接即可

Others

网站的稳定性是网站权重的一大标准，一个经常打不开的网站是不能被赋予很高的权重的！
保障您的网站安全，被黑客入侵会降低您的权重
如果您做了什么违规的事情，百度有可能会将您拉黑
如果进入黑名单，您的权重将会达到很低的水平，甚至清除收录！
出现以下现象，可能会被列入黑名单

  • 出现大量垃圾链接
  • 文章异常

友链

定义

友情链接是您和您的朋友的网站互通的方式，爬虫也会检查友情链接、

好处

- 提高权重
- 增加流量
- 提高关键词排名
- 增加出站链接
- 提高知名度
- 提高爬虫率

注意事项

- 权重相似
- 不和违规网站交换
- 尽量使用锚文本，少使用图片
- 要是活站，有人维护！

反链

定义

反向链接是指您文章中指向您站点的链接
反向链接是衡量站点复杂程度的一个指数

GitHub pages问题

  • 起因

    • 百度爬虫会被github pages禁止访问
    • github承认此问题

  • 解决方案

    • 使用其他服务
    • 给网站套CDN

    使用CDN能提高用户体验且不影响权重
